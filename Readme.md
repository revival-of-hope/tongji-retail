## 项目介绍
本系统以电商平台为业务背景，采用前后端分离架构（C# ASP.NET Core 9 + EntityFrameworkCore 9;  Next.js 14 + Shadcn UI组件库 Oracle 21c）。系统涵盖顾客、商家、管理员、客服四大角色。使用docker compose进行部署

- 前后端的需求已经分别给在各自的文件夹Readme.md中.
