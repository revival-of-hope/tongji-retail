# 商品零售管理系统 - 数据库设计文档

## 1. 引言

### 1.1 编写目的
本文档详细描述了“商品零售管理系统”的数据库结构设计，包括实体关系（E-R）模型、表结构定义及主外键约束。本设计基于 Oracle 21c 数据库，并使用 Entity Framework Core 进行对象关系映射（ORM）。

### 1.2 设计原则
- **第三范式（3NF）**：消除传递依赖，确保数据冗余最小化。
- **扩展性**：支持多角色、多店铺、多分类层级。
- **数据完整性**：使用外键约束保证关联数据的有效性，关键金额字段使用 `DECIMAL(18,2)` 确保精度。
- **审计追踪**：所有核心表均包含 `CreatedAt` 和 `UpdatedAt` 字段。

## 2. 概念结构设计 (E-R 模型)

系统包含以下核心实体及其关系：

1. **用户 (User)**：系统的基础账号，包含顾客、商家、管理员、客服四种角色。
2. **商家 (Merchant)**：由用户申请成为，拥有店铺信息。一个用户最多对应一个商家。
3. **商品分类 (Category)**：支持无限层级的树形结构（父子关系）。
4. **商品 (Product)**：由商家发布，属于某个分类。一个商家可发布多个商品。
5. **商品图片 (ProductImage)**：一个商品可包含多张图片，其中一张为主图。
6. **购物车 (ShoppingCart)**：每个用户拥有一个购物车。
7. **购物车明细 (CartItem)**：购物车中包含多个商品项及数量。
8. **订单 (Order)**：用户提交的购买记录，包含总金额、收货地址等。
9. **订单明细 (OrderItem)**：订单中包含的具体商品、数量及下单时的单价（价格快照）。
10. **支付记录 (Payment)**：订单的支付流水，记录支付方式和金额。
11. **商品评价 (ProductReview)**：用户对已完成订单中的商品进行的评分和文字评价。
12. **客服工单 (CustomerServiceTicket)**：用户提交的售后问题，分配给特定客服处理。

## 3. 逻辑结构设计 (表结构)

本系统共设计了 12 张核心数据表，具体定义如下：

### 3.1 用户表 (Users)
存储所有角色的登录账号信息。
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| Username | VARCHAR2(50) | UNIQUE, NOT NULL | 用户名 |
| PasswordHash | VARCHAR2(255) | NOT NULL | BCrypt 加密密码 |
| Email | VARCHAR2(100) | UNIQUE | 邮箱 |
| Phone | VARCHAR2(20) | | 手机号 |
| Role | NUMBER(10) | NOT NULL | 角色：0=Admin, 1=Customer, 2=Merchant, 3=CustomerService |
| IsActive | NUMBER(1) | NOT NULL | 是否启用（1=是，0=否） |
| CreatedAt | TIMESTAMP | NOT NULL | 注册时间 |

### 3.2 商家表 (Merchants)
存储店铺信息及入驻审核状态。
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| UserId | NUMBER(19) | FK(Users.Id), UNIQUE | 关联用户 |
| StoreName | VARCHAR2(100) | NOT NULL | 店铺名称 |
| Description | VARCHAR2(500) | | 店铺描述 |
| Status | NUMBER(10) | NOT NULL | 状态：0=待审核, 1=已批准, 2=已拒绝 |
| CreatedAt | TIMESTAMP | NOT NULL | 申请时间 |

### 3.3 商品分类表 (Categories)
支持多级分类。
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| ParentId | NUMBER(19) | FK(Categories.Id) | 父分类ID（可空） |
| Name | VARCHAR2(50) | NOT NULL | 分类名称 |
| SortOrder | NUMBER(10) | NOT NULL | 排序号 |

### 3.4 商品表 (Products)
存储商品核心信息。
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| MerchantId | NUMBER(19) | FK(Merchants.Id) | 所属商家 |
| CategoryId | NUMBER(19) | FK(Categories.Id) | 所属分类 |
| Name | VARCHAR2(200) | NOT NULL | 商品名称 |
| Description | CLOB | | 商品详情描述 |
| Price | DECIMAL(18,2) | NOT NULL | 当前售价 |
| StockQuantity | NUMBER(10) | NOT NULL | 当前库存量 |
| SoldCount | NUMBER(10) | NOT NULL | 累计销量 |
| AvgRating | DECIMAL(3,2) | NOT NULL | 平均评分（0.00-5.00） |
| ReviewCount | NUMBER(10) | NOT NULL | 评价总数 |
| Status | NUMBER(10) | NOT NULL | 状态：0=待审核, 1=已上架, 2=已下架, 3=已拒绝 |
| CreatedAt | TIMESTAMP | NOT NULL | 发布时间 |

### 3.5 商品图片表 (ProductImages)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| ProductId | NUMBER(19) | FK(Products.Id) | 所属商品 |
| ImageUrl | VARCHAR2(500) | NOT NULL | 图片链接 |
| IsMain | NUMBER(1) | NOT NULL | 是否为主图（1=是，0=否） |
| SortOrder | NUMBER(10) | NOT NULL | 排序号 |

### 3.6 购物车表 (ShoppingCarts)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| UserId | NUMBER(19) | FK(Users.Id), UNIQUE | 所属用户 |
| UpdatedAt | TIMESTAMP | NOT NULL | 最后更新时间 |

### 3.7 购物车明细表 (CartItems)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| CartId | NUMBER(19) | FK(ShoppingCarts.Id) | 所属购物车 |
| ProductId | NUMBER(19) | FK(Products.Id) | 所属商品 |
| Quantity | NUMBER(10) | NOT NULL | 加购数量 |

### 3.8 订单表 (Orders)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| UserId | NUMBER(19) | FK(Users.Id) | 下单用户 |
| OrderNo | VARCHAR2(50) | UNIQUE, NOT NULL | 订单编号（业务流水号） |
| TotalAmount | DECIMAL(18,2) | NOT NULL | 订单总金额 |
| Status | NUMBER(10) | NOT NULL | 状态：0=待支付, 1=待发货, 2=已发货, 3=已完成, 4=已取消 |
| ShippingAddress | VARCHAR2(500) | NOT NULL | 收货地址 |
| Remark | VARCHAR2(500) | | 订单备注 |
| ExpireAt | TIMESTAMP | NOT NULL | 支付超时时间 |
| CreatedAt | TIMESTAMP | NOT NULL | 下单时间 |

### 3.9 订单明细表 (OrderItems)
记录下单时的价格快照。
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| OrderId | NUMBER(19) | FK(Orders.Id) | 所属订单 |
| ProductId | NUMBER(19) | FK(Products.Id) | 所属商品 |
| Quantity | NUMBER(10) | NOT NULL | 购买数量 |
| UnitPrice | DECIMAL(18,2) | NOT NULL | 下单时的单价（价格快照） |
| SubTotal | DECIMAL(18,2) | NOT NULL | 小计金额（UnitPrice * Quantity） |

### 3.10 支付记录表 (Payments)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| OrderId | NUMBER(19) | FK(Orders.Id), UNIQUE | 关联订单 |
| Amount | DECIMAL(18,2) | NOT NULL | 实际支付金额 |
| PaymentMethod | NUMBER(10) | NOT NULL | 支付方式：0=支付宝, 1=微信, 2=信用卡 |
| Status | NUMBER(10) | NOT NULL | 状态：0=待支付, 1=支付成功, 2=支付失败 |
| TransactionId | VARCHAR2(100) | | 第三方交易流水号 |
| PaidAt | TIMESTAMP | | 支付完成时间 |

### 3.11 商品评价表 (ProductReviews)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| ProductId | NUMBER(19) | FK(Products.Id) | 评价的商品 |
| UserId | NUMBER(19) | FK(Users.Id) | 评价人 |
| OrderId | NUMBER(19) | FK(Orders.Id) | 关联订单 |
| Rating | NUMBER(10) | NOT NULL | 评分（1-5星） |
| Comment | VARCHAR2(1000) | | 评价内容 |
| CreatedAt | TIMESTAMP | NOT NULL | 评价时间 |

### 3.12 客服工单表 (CustomerServiceTickets)
| 字段名 | 数据类型 | 约束 | 说明 |
|---|---|---|---|
| Id | NUMBER(19) | PK | 主键，自增 |
| UserId | NUMBER(19) | FK(Users.Id) | 提交人（顾客） |
| OrderId | NUMBER(19) | FK(Orders.Id) | 关联订单（可空） |
| AssignedTo | NUMBER(19) | FK(Users.Id) | 分配的客服人员ID（可空） |
| Subject | VARCHAR2(200) | NOT NULL | 工单主题 |
| Description | VARCHAR2(2000) | NOT NULL | 问题描述 |
| Reply | VARCHAR2(2000) | | 客服回复内容 |
| Status | NUMBER(10) | NOT NULL | 状态：0=待处理, 1=处理中, 2=已解决, 3=已关闭 |
| CreatedAt | TIMESTAMP | NOT NULL | 提交时间 |
| UpdatedAt | TIMESTAMP | NOT NULL | 最后更新时间 |
